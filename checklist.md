## 폴더 구조

### `/class` - 강의 자료
- 주차별 강의 노트 및 자료
- 예: week-01.md, week-02.md, ...

### `/practice` - 실습 코드
- 챕터별 실습 코드
- 예: chapter1/, chapter2/, ..., chapter12/

### `/code` - 프로젝트 코드
- 주요 애플리케이션 코드
- 에이전트 및 워크플로우 구현

### `/data` - 데이터
- 입력 데이터
- 처리 결과
- 데이터베이스 파일

### `/output` - 출력 결과
- 실행 결과 저장
- 로그 및 리포트

### `/logs` - 로그 파일
- 애플리케이션 로그
- 에러 로그

### `/docs` - 문서
- 프로젝트 문서
- API 문서
- 아키텍처 가이드
